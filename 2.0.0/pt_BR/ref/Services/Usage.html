Service Usage

A service contains business logic that can be reused across a Grails application. In Grails a service is a class with a name that ends in the convention "Service" and lives in the grails-app/services directory. A service can be created with the commandLine command:

grails create-service org.bookstore.Book

or with your favourite IDE or text editor.

package org.bookstore
class BookService {
    Book[] getBooks() {
        Book.list() as Book[]
    }
}

Refer to the user guide topic on guide:services for more information.

Uso dos Services

Um service deve contar inteligência e lógica de negócio que poderão ser reaproveitadas em mais de um ponto de sua aplicação Grails. Para uma classe ser um Service ela deve obedecer a convenção de ter seu nome terminado em "Service" E estar no diretório grails-app/services. Um service pode ser criado usando o comando commandLine:

grails create-service org.bookstore.Book

ou então através de sua IDE ou editor de texto preferido.

package org.bookstore
class BookService {
    Book[] getBooks() {
        Book.list() as Book[]
    }
}

Consulte o Guia do usuário na seção guide:services para mais informações.
